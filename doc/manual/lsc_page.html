Research Projects

The Laboratory for Scientific Computing (LSC) has a number of on-going projects in various areas of high-performance computing for scientific and engineering applications.



Generic Programming

  • MTL : The Matrix Template Library
  • ITL : The Iterative Template Library
  • GGCL : The Generic Graph Component Library

High-Performance Object-Oriented Scientific Computing

  • IML++ : an iterative methods library in C++
  • MPI-2 C++ : Bindings which implement the C++ interface to MPI.
  • OOMPI : Object oriented extensions to the MPI message passing standard.
  • Sparselib++ : a C++ class library for portable and efficient sparse matrix computations

MPI Parallel Computing

  • LAM / MPI : Public domain implementation of the MPI (Message Passing Interface) standard.
  • MPI-2 C++ : Bindings which implement the C++ interface to MPI.
  • OOMPI : Object oriented extensions to the MPI message passing standard.
  • MPI tutorials : Tutorials offered by the University of Notre Dame on MPI.
  • IMPI Server : Interoperable MPI Server Software

Applications

  • Three-Dimensional Aeroacoustic Simulation : Aerodynamics and Aeroacoustics of Unsteady Rotational Flows

Image Processing and Robotic Vision

  • Maximum Likelihood Parameter Estimation for Non-Gaussian Prior Signal Models : Image and Video Analysis
  • PIPT : Parallel Image Processing Toolkit

Irreproducible Results

  • DomeCam!
  • ORCA
